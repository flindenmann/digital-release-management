- [x] Implementierung von Sammeltasks. Sammeltasks sind eine Sammlung von Tasks, dessen Start und Ende durch die enthaltenen Tasks definiert wird. Ein Task kann nur einem einzigen Sammeltask zugewiesen werden. In der Taskliste muss man einen Sammeltask auf- und zuklappen können. Dabei werden die zugehörigen Tasks angezeigt, bzw. ausgeblendet. Erweitere den Task Dialog mit einer Checkbox unterhalb der Meilenstein-Checkbox, mit der man einen Sammeltask definieren kann. Meilenstein und Sammeltask-Checkboxen können nicht gleichzeitig ausgewählt sein. Die Eingabefelder für einen Sammeltask sind gleich wie für einen Tasks. Allerdings kann man Start, Ende und Dauer nicht editieren. In der Taskliste werden im Sammeltask der früheste Start und das späteste Ende als Start und Ende angezeigt. Die Applikation und der Status werden nicht angezeigt, das sie pro Task unterschiedlich sein können. In der Taskliste werden Sammeltasks hellgrau hinterlegt angezeigt.
